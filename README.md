# Arabic Anime Streaming Site

A modern anime streaming website with Arabic interface built with Next.js 14.

## Features
- Arabic RTL interface
- Responsive design
- Supabase backend
- Tailwind CSS styling

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Database:** Supabase
- **Fonts:** Cairo & Tajawal (Arabic fonts)
